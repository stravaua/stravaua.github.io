Детальна інструкція із створення запиту в техпідтримку Strava

Заповнювати найкраще англійською мовою. Можна скористатися ChatGPT для допомоги.

  1. Увійдіть на сторінці Strava Support "Submit a Request".
  2. Виберіть "Feature Requests" в меню "Choose a category".
  3. В полі "Subject" вкажіть "Add Ukrainian Language".
  4. В полі "Description" опишіть, чому важливо додати українську мову. Приклад тексту, який можна підправити за допомогою ChatGPT:

    Hello Strava Team,

    I would like to request the addition of Ukrainian language support to the Strava app.

    Ukrainian is the native language of millions of users, and adding it would significantly improve accessibility and user experience for the Ukrainian-speaking community.

    There has already been a similar request in the past that gained substantial community support, gathering over 1,900 votes: https://communityhub.strava.com/ideas/adding-the-ukrainian-language-to-the-application-6861

    Adding Ukrainian would help make Strava more inclusive and welcoming to a large and active group of athletes.

    Thank you for considering this request.

  5. В полі "Product Area" виберіть "Other".
  6. В полі "Platform" виберіть "Both mobile & web".
  7. Натисніть "Submit". Вам на пошту прийде підтвердження про отримання запиту.

Ви чудові, дякуємо за вашу підтримку!

Як налаштувати Strava на англійську

Якщо ви знаєте англійську мову, налаштуйте додаток і вебсайт Strava так, щоб інтерфейс відображався цією мовою.

Якщо не знаєте англійську, читайте наступний розділ.

Для цього потрібно виконати три кроки:

  1. Змінити мову додатку (iOS, Android).
  2. Змінити мову веб-інтерфейсу.
  3. Змінити мову, яку Strava використовує для назв активностей.

1. Як змінити мову додатку

Як змінити мову в iOS

Спочатку додайте англійську мову по цій інструкції. Далі виконайте наступне:

  1. Відкрийте програму "Параметри" (Settings)
  2. Натисніть "Програми" (Apps).
  3. Виберіть додаток "Strava".
  4. У "Пріоритетна мова" (Preferred Language) виберіть "English".

Важливо вибрати саме англійську. Якщо встановити українську, то Strava відображатиметься російською.

Як змінити мову в Android

Дійте за цією інструкцією:

  1. Відкрийте додаток "Налаштування" (Settings)
  2. Натисніть "Система" (System) > "Мови" (Languages) > "Мови додатків" (App Languages).
  3. Виберіть додаток "Strava".
  4. Виберіть "English".

2. Як змінити мову веб-інтерфейсу

  1. Увійти в Strava в браузері з комп'ютера.
  2. Долистати вниз, у футері праворуч вибрати мову "English (US)" або "British English".

3. Як змінити мову, якою Strava називає активності

Щоб змінити мову, яку Strava використовує для автоматичних назв активностей, потрібно змінити мову додатку або веб-інтерфейсу.

Як користуватись Strava, якщо не знаєте англійську

Якщо ви не знаєте англійську, ви можете використовувати вбудований в браузер Chrome перекладач або завчити кілька англійських слів.

Використання вбудованого в Chrome перекладача

Перекладач в десктопному браузері Chrome

Щоб Chrome автоматично перекладав Strava з англійської, потрібно зробити таке:

  1. Переконатись, що браузер відображається українською.
  2. Увійти в Strava.
  3. Натиснути в браузері праворуч вгорі "Налаштування" - "Перекласти", вибрати "Українська".
  4. Додатково в меню перекладу праворуч вибрати галочку "завжди перекладати з такої мови: англійська".

У результаті всі сторінки Strava автоматично перекладатимуться українською за допомогою браузера:

Перекладач в мобільному браузері Chrome (iOS)

Щоб Chrome в iOS автоматично перекладав Strava з англійської, потрібно зробити таке:

  1. Переконатись, що браузер відображається українською.
  2. Увійти в Strava з мобільного браузера. Якщо після входу перенаправляє в додаток Strava, відкрийте strava.com в браузері ще раз.
  3. Натиснути в браузері праворуч внизу "Налаштування" - "Перекласти", вибрати "Українська".

Як додати карту Support Ukraine

Наприкінці березня 2022 року Strava додала карту Support Ukraine, щоб підтримати Україну.

Але в листопаді 2025 року ця та інші типи карт зникли з додатку та сайту Strava. Вони були замінені на "Causes". Додатково Strava дозволила вставляти посилання на сторінки підтримки справи в опис активності. Але ж ми хочемо бачити карту "Support Ukraine" на своїх маршрутах!

На щастя, є можливість додавати карту Support Ukraine програмно через API. Станом на кінець листопада 2025 року карти продовжують відображатися в додатку та на сайті.

Як встановити карту "Support Ukraine" через ActivityFix

Існує сторонній веб-сайт ActivityFix, який дозволяє редагувати активності в Strava залежно від різних параметрів. Під капотом використовується Strava API. Алгоритм такий:

  1. Увійдіть в ActivityFix через ваш акаунт Strava.
  2. Ліворуч на панелі "Dashboard" виберіть "Add Rule".
  3. Введіть наступні параметри для правила:
    • What would you like to call this rule: Use "Support Ukraine" map for all activities.
    • What should be changed: Change map type to Support Ukraine.
    • When should it be changed: Activity type is Any Activity.
    • If the rule matches: Yes: Continue processing remaining rules.
  4. Натисніть "Save Rule". Його буде додано в список правил "My Rules".
  5. Опціонально: можна перевірити правило на одній активності, натиснувши "Test Rule".

Після цього всі нові активності, які ви завантажите в Strava, автоматично отримуватимуть карту "Support Ukraine".

Існує можливість застосувати правило до наявних активностей, натиснувши "Process Old Activities". Дивіться інструкцію на YouTube ActivityFix.

Схожим чином можна вставляти карту Support Ukraine через інші сторонні сервіси, наприклад, Strautomator.

Як встановити карту "Support Ukraine", використовуючи Strava API

Якщо ви вмієте програмувати, можна додати карту "Support Ukraine" через Strava API, використовуючи Update Activity ендпоінт. Алгоритм наступний:

  1. Отримуємо токен доступу по інструкції.
  2. Виконуємо PUT запит на ендпоінт https://www.strava.com/api/v3/activities/{activity_id} з JSON тілом запиту {"selected_polyline_style":"ukraine"}.
curl -X PUT "https://www.strava.com/api/v3/activities/{activity_id}" \
  -H "Authorization: Bearer {access_token}" \
  -H "Content-Type: application/json" \
  -d '{"selected_polyline_style":"ukraine"}'

У запиті {access_token} - токен доступу з першого кроку, {activity_id} - ідентифікатор активності.

Параметр selected_polyline_style не задокументований і може змінитися в майбутньому. Всі його значення описані тут.
